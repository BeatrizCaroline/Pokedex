# Pokédex

Uma Pokédex interativa e responsiva que consome a [PokeAPI](https://pokeapi.co/) para exibir informações sobre os Pokémon. Este projeto foi desenvolvido com HTML5, CSS3 e JavaScript puro, oferecendo uma experiência simples e funcional para os fãs da franquia.

## ✨ Funcionalidades

- **Listagem de Pokémon:** Exibe uma lista de Pokémon com seus nomes, tipos e imagens.
- **Carregamento Dinâmico:** Botão "Load More" para carregar mais Pokémon gradualmente.
- **Pesquisa:** Campo de pesquisa para filtrar Pokémon por nome.
- **Design Responsivo:** Interface adaptável para diferentes tamanhos de tela.
- **Cores por Tipo:** Cada card de Pokémon possui uma cor de fundo baseada no seu tipo primário.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação do conteúdo.
- **CSS3:** Estilização e layout responsivo (com auxílio do Normalize.css e fonte Roboto).
- **JavaScript (ES6):** Lógica de consumo da API, manipulação do DOM e filtros.
- **PokeAPI:** Fonte de dados pública e gratuita sobre Pokémon.
